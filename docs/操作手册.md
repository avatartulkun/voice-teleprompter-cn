# 语音跟读提词器操作手册

适用版本：v1.3.0

语音跟读提词器是一款面向中文用户的移动提词工具，适合演讲、录课、短视频口播、直播话术和朗读练习。应用支持稿件导入、语音跟读识别、已读内容自动高亮、横屏提词、自动滚动和显示样式调整。

## 1. 安装应用

1. 在 GitHub Releases 页面下载 APK 文件：

```text
voice-teleprompter-cn-android-v1.3.0-release.apk
```

2. 将 APK 安装到 Android 手机。
3. 如果系统提示“禁止安装未知来源应用”，请按手机系统提示允许当前浏览器或文件管理器安装 APK。
4. 安装完成后，打开“语音跟读提词器”。

## 2. 首页三步准备

打开应用后，首页会显示三个主要步骤。

### 第一步：设置识别密钥

点击“设置”，先选择语音服务商，再按弹窗中的字段提示填写。不同平台的名称不完全一样，例如：

```text
百度智能云：AppID、API Key
腾讯云：AppID、SecretId、SecretKey
阿里云：AppKey、Token
```

当前版本保留百度智能云、腾讯云和阿里云三种实时语音识别配置，便于测试和维护。

如果不知道如何获取密钥，可以在设置弹窗底部查看“如何获取密钥”说明，并点击“查看实时语音识别开通说明”。

可参考的平台包括：

- 百度智能云
- 腾讯云
- 阿里云

开通或购买时，请选择“实时语音识别”。这些平台通常都有免费额度，对于轻中量用户基本可以满足日常需求。更详细步骤可参考：

[实时语音识别开通说明](https://github.com/avatartulkun/voice-teleprompter-cn/blob/main/docs/%E5%AE%9E%E6%97%B6%E8%AF%AD%E9%9F%B3%E8%AF%86%E5%88%AB%E5%BC%80%E9%80%9A%E8%AF%B4%E6%98%8E.md)

建议使用谷歌浏览器打开该说明。

### 第二步：测试麦克风

点击“麦克风”，检查手机是否已允许麦克风权限。

如果系统弹出权限请求，请选择允许。语音跟读识别需要麦克风权限才能正常工作。

### 第三步：导入与设置

第三步包含三个入口：

```text
稿件
跟读设置
显示设置
```

你也可以先点击“进入提词”，进入提词后再调整这些设置。

## 3. 编辑或导入稿件

点击首页的“稿件”，进入稿件编辑弹窗。

你可以：

- 输入演讲稿、口播稿、录课稿或朗读文本
- 修改已有稿件
- 保存当前稿件

建议稿件尽量使用清晰的中文标点和自然分段，这样阅读时更容易跟上节奏。

## 4. 进入提词模式

准备完成后，点击首页中间的圆形按钮“进入提词”。

首次进入时，应用会弹出“关于与协议”：

1. 阅读用户协议和隐私政策。
2. 勾选“已阅读并同意用户协议和隐私政策”。
3. 点击“同意并进入”。

应用会记住你的选择，之后再次进入提词时不会重复弹出。

## 5. 开始跟读

进入提词模式后，点击“开始”。

应用会启动语音识别。你开始朗读稿件后，提词区会根据识别结果逐步高亮已读内容，并提示当前朗读位置。

常用按钮：

```text
开始：开始语音跟读
暂停：暂停识别或朗读
重置：回到稿件开头
后退：将当前朗读位置向前调整
前进：将当前朗读位置向后调整
滚动：开启自动滚动
停止滚动：关闭自动滚动
```

## 6. 跟读设置

点击“跟读设置”，可以调整跟读时的行为。

常见设置包括：

- 变色速度
- 卡住时自动推进
- 滑动速度

如果发现高亮速度太快或太慢，可以先调整“变色速度”。如果朗读时停顿较多，可以适当调整“卡住时自动推进”。

## 7. 显示设置

点击“显示设置”，可以调整提词界面的视觉效果。

可调整内容包括：

- 字体大小
- 已读颜色
- 当前字颜色
- 底色

如果用于演讲或录课，建议先根据手机支架距离调整字体大小。距离较远时，把字体调大一些会更舒服。

## 8. 横屏和全屏

提词模式支持横屏和全屏显示。

常用入口：

```text
横/竖：切换横屏或竖屏
全屏：进入全屏提词
退出全屏：回到普通提词界面
返回：回到首页
```

横屏更适合演讲、录课和手机支架场景；竖屏更适合手持练习和短稿件。

## 9. 邮件反馈

首页右下角有邮箱图标。

点击后，系统会尝试打开手机上的邮件应用，并自动填写反馈邮件的主题和基础内容。

联系方式：

```text
tulkun@foxmail.com
```

如果手机没有可用邮件应用，也可以手动复制该邮箱发送反馈。

## 10. 关于与协议

首页底部有“关于与协议”入口。

这里可以查看：

- 当前版本号
- 开源地址
- 用户协议
- 隐私政策
- 联系方式

## 11. 隐私说明

本软件不收集、保存或上传任何个人隐私信息。

应用需要麦克风权限，用于语音跟读识别。当你使用云端语音识别功能时，语音识别会连接你自行配置的第三方语音识别服务。

请不要在稿件、朗读内容或识别内容中包含以下信息：

- 身份证号
- 手机号
- 家庭地址
- 账号密码
- 财务信息
- 其他个人隐私或敏感信息

## 12. 常见问题

### 识别没有反应怎么办？

请检查：

- 是否已允许麦克风权限
- 手机网络是否正常
- 所选服务商的识别密钥字段是否填写正确
- 是否开通了“实时语音识别”
- 免费额度是否已用完
- 账号是否欠费或服务是否被暂停

### 识别密钥应该怎么填？

不同服务商字段不同，请按设置弹窗提示填写：

- 百度智能云：AppID、API Key
- 腾讯云：AppID、SecretId、SecretKey
- 阿里云：AppKey、Token

腾讯云的 SecretKey 通常只在创建密钥时完整显示一次，请创建后及时保存。阿里云 Endpoint 可不填，默认使用应用内置地址。

### 为什么要选择“实时语音识别”？

本软件的跟读高亮需要边朗读边识别，实时语音识别更适合这种使用方式。不要误选“实时语音转写”等不对应的服务。

### 上传 GitHub 时要不要上传 APK？

源码仓库里不建议提交 APK。APK 建议单独上传到 GitHub Releases 页面。

## 13. 推荐使用流程

第一次使用可以按下面顺序操作：

1. 安装 APK。
2. 打开应用。
3. 点击“设置”，选择服务商并填写对应识别密钥。
4. 点击“麦克风”，允许并测试麦克风权限。
5. 点击“稿件”，输入或粘贴你的稿件。
6. 点击“进入提词”。
7. 首次进入时阅读并同意协议。
8. 点击“开始”，开始朗读。
9. 根据需要调整跟读设置和显示设置。

完成以上步骤后，就可以把它作为日常演讲、录课和口播练习提词器使用。
