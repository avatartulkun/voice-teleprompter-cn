# 实时语音识别开通说明（以百度语音技术为例）

本文档用于说明语音跟读提词器 Android 版所需的实时语音识别服务开通和配置流程。

本文以百度语音技术中的“实时语音识别”为例，说明如何开通服务、获取密钥并在应用中完成配置。对外介绍应用时，仍建议使用“云端语音识别”“实时语音识别服务”等通用名称。

## 1. 登录控制台

登录百度智能云或百度 AI 开放平台控制台。

如果账号未完成实名认证，需要先按平台提示完成认证，否则可能无法创建应用、开通服务或使用付费能力。

## 2. 进入语音技术服务

在控制台中找到“人工智能”或“AI 服务”分类，进入“语音技术”相关产品页面。

在百度语音技术的语音识别能力中，选择：

```text
实时语音识别
```

不要选择“实时语音转写”相关服务。

## 3. 创建应用

在语音技术控制台中新建应用。

应用类型可按实际项目选择移动应用、Android 应用或通用应用。

应用名称可以填写：

```text
语音跟读提词器
```

创建完成后，进入应用详情页。

## 4. 获取识别密钥

在应用详情中记录以下信息：

```text
AppID
API Key
Secret Key（可不用填）
```

这些信息用于应用内“设置识别密钥”页面。

请注意：

- 不要把真实密钥提交到 GitHub。
- 不要把真实密钥写入公开 README。
- 如果需要截图说明，请遮挡密钥内容。

## 5. 开通实时语音识别能力

在应用能力或产品服务配置中，确认已启用实时语音识别相关能力。

如果控制台要求领取免费额度、开通计费或同意服务协议，需要按页面提示完成。

如果项目同时使用普通语音识别作为兜底，也需要确认普通语音识别能力可用。

## 6. 费用说明

截至 2026 年 6 月 23 日，实时语音识别通常提供：

```text
用户开通后有每月 10 小时免费额度，特别提醒的是每月，而非首月，基本能满足日常使用。
```

如果免费额度不够，可按量付费使用，目前价格为：

```text
3 元/小时
```

以上额度与价格仅作为开发记录，最终价格、免费额度和计费规则以官方控制台与官方价格页为准。

## 7. 在应用中配置并测试

1. 安装最新 APK。
2. 打开应用。
3. 进入“设置识别密钥”。
4. 填写 AppID 和 API Key。
5. 返回首页，先执行“麦克风测试”。
6. 进入提词模式。
7. 点击“开始”，朗读稿件内容。

如果识别正常，提词区会逐步高亮已读文字。

## 8. 常见检查项

- 手机已允许麦克风权限。
- 手机网络正常。
- AppID 应填写数字形式。
- AppID 和 API Key 不要填反。
- 控制台中应用状态正常。
- 实时语音识别能力已开通。
- 账号没有欠费。
- 免费额度没有用完，或已开通按量计费。
